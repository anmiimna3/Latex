        Frist suppose that there exists $a \ne b$ such that $v(x - a) \ne 0$ and $v(x - b) \ne 0$. Then we have:
        \begin{gather*}
            v(ax - ab) = v(a) + v(x - b) = v(x - b) \ne 0 \\
            v(bx - ba) = v(b) + v(x - a) = v(x - a) \ne 0
        \end{gather*}
        Then we can write:
        \begin{gather*}
            0 = v(b - a) = v((x - a) - (x - b))  \ge \min \{v(x - a), v(x - b) \}
        \end{gather*}
\end{enumerate}
