$\lambda$-ترم‌های $O$ و $E$ را چنان تعریف کنید که 
\begin{gather*}
    OC_m =_\beta \begin{cases}
        true \ \ \ \ \ If \ m \ is \ odd \\
        false \ \ \ O.W.
    \end{cases} 
    ,
    EC_m =_\beta \begin{cases}
        true \ \ \ \ \ If \ m \ is \ even \\
        false \ \ \ O.W.
    \end{cases}
\end{gather*}
